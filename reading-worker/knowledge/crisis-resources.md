# Krisen- und Hilfsangebote (DACH)

Wenn ein Reading sensible Themen berührt (Trauma, Suizidalität, sexueller Missbrauch, Gewalt, akute Lebenskrise, Schuldenkrise), nenne im Reading **immer** mindestens eine passende Anlaufstelle aus dieser Liste. Coaching ersetzt keine Therapie und keine Krisenintervention.

## Allgemeine 24/7-Krisenhilfe

**Deutschland:**
- Telefonseelsorge (evangelisch): **0800 111 0 111** (kostenlos, 24/7, anonym)
- Telefonseelsorge (katholisch): **0800 111 0 222** (kostenlos, 24/7, anonym)
- Telefonseelsorge Chat & Mail: online.telefonseelsorge.de
- Bei akuter Suizidalität / medizinischem Notfall: **112**
- Sozialpsychiatrischer Krisendienst (regional, nicht überall 24/7): über 116 117 erfragen

**Österreich:**
- Telefonseelsorge: **142** (kostenlos, 24/7)
- Notruf: 112

**Schweiz:**
- Die Dargebotene Hand: **143** (kostenlos, 24/7)
- Notruf: 112 / 144 (Sanität)

## Coach-Anweisung für das Reading

Wenn das Reading **eines dieser Themen** berührt, baue einen kurzen, ruhigen Hinweis ein:

> "Wenn dich dieses Thema gerade akut belastet, sind diese Stellen rund um die Uhr für dich da: Telefonseelsorge 0800 111 0 111 (Deutschland) / 142 (Österreich) / 143 (Schweiz). Das Reading ersetzt keine therapeutische Begleitung."

Bei spezifischen Themen (Missbrauch, Gewalt, Schulden) zusätzlich die passende Spezialhotline nennen — keine Pflicht-Liste, sondern den 1–2 thematisch wirklich passenden Eintrag.
